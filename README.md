## About

[Viewing Party](https://intense-harbor-71203.herokuapp.com/) is an application in which users can explore movie options and create a viewing party event for the user and their friends.

User stories were created by [Turing School](https://github.com/turingschool-examples/viewing_party).

## Initial Setup
1. Fork the Project
2. Set up the rails database ```rails db:{create,migrate}```
3. Create ruby file to securely store API keys. We chose Figaro, but you do you!
4. Add the following key to your secured file:  
    *  *Movie DB key*   [request MovieDB API key](https://developers.themoviedb.org/3/getting-started/introduction)     
5. Plan a party!
